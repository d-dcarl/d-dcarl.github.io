Virtual Educator Experience (VEex/VREDSIM)

C# Unity 3D

Overview

In Virtual Educator Experience (VEex/VREDSIM) you are trained as a teacher through classroom and relaxing simulations to learn to mitigate classroom behavior and learn stress management techniques.

My Role

Client based project through the Games and Apps Lab (GApp Lab) at the University of Utah.

  • Used UnityVR to prototype, playtest, and manage project
  • Used Jira, Google Drive, and Microsoft Teams to manage the project team, schedule, and client interactions
  • Setup and ran team and client meetings and presentations
  • Coordinated with clients, project managers, and scheduling throughout the project lifetime
  • Produced and crafted trailers, GDD, Game Project SDK, branding, project memos, and other critical documents throughout project lifetime

Tech Stack

  • Unity VR
  • C#
  • Maya
  • Google AI
  • Pro Tools
  • Adobe Illustrator

Challenge & Solution

The biggest challenge was coordinating with the team to create the game. I had to learn how to use Unreal Engine 4 and how to use the tools to create the game.

Another challenge was balancing difficulty progression as players advanced through the dungeon. I implemented a dynamic difficulty scaling system that adjusted based on player performance.

The solution was to use Unreal Engine 4 to playtest and iterate on the game. I used Pro Tools to create the music and sound effects. I used Trello and ClickUp to manage the project. I coordinated with the team to create the game.

Results

The prototype was well-received at the end of the year, with players particularly enjoying the main mechanic and general atmosphere.
